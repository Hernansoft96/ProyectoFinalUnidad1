# Proyecto web - Lavadero de vehiculos "Detalle De Elite"

## INTRODUCCION

El presente proyecto tiene como propósito el desarrollo de un sitio web responsive para un lavadero de vehículos llamado **“Detalle De Elite”**.

La idea principal es crear una plataforma digital moderna que permita presentar de forma clara los servicios ofrecidos, mostrar precios en formato prototipo **(sujetos a cambios según costos reales)** y facilitar el contacto con los clientes, incluyendo su ubicación.

El sitio busca transmitir una imagen profesional, organizada y atractiva, adaptándose a diferentes dispositivos.

---

## OBJETIVO 

Desarrollar un sitio web funcional y responsive que permita aplicar los conocimientos adquiridos durante el proceso de formación.

Se busca que la página permita visualizar de manera clara los servicios de lavado de carros y motos, brindando una experiencia de usuario sencilla, intuitiva y accesible.

---

## TECNOLOGIAS UTILIZADAS

- **HTML5** = Estructura del sitio web 
- **CSS3** = Estilos y diseños 
- **Bootstrap** = Diseño responsive y componentes modernos  

---

## ESTRUCTURA DEL SITIO

El proyecto está compuesto por dos páginas principales:

### index.html

- Presentación del negocio  
- Sección de servicios  
- Contenido multimedia (imágenes / carousel)  

### pagina.html

- Precios (prototipo)  
- Beneficios del servicio  
- Formulario de contacto  

---

## CARACTERISTICAS

- Diseño responsive  
- Uso de Bootstrap  
- Navegación entre páginas  
- Carousel de imágenes  
- Tarjetas de servicios  
- Formulario de contacto  
- Interfaz clara y organizada  

---

## CONCLUSION

El desarrollo de este proyecto permitió aplicar conocimientos fundamentales de diseño y desarrollo web, logrando un sitio funcional, organizado y adaptable a diferentes dispositivos.

Además, representa una base sólida para la futura implementación del emprendimiento **“Detalle De Elite”**, con posibilidades de mejora y escalabilidad.
